## Overview

The XtendOps API allows you to programmatically access SmartAgent capabilities, enabling you to integrate AI-powered customer support into your existing platforms and workflows.

## API Base URL

All API endpoints are available at:

```
https://agent-assist.xtendops.com/api
```

## Authentication

All API requests require authentication using a Bearer token passed in the `Authorization` header:

```bash
Authorization: YOUR_API_TOKEN
```

## Request Format

All API requests should:
- Use HTTPS
- Include `Content-Type: application/json` header
- Include your authentication token in the `Authorization` header
- Send data in JSON format

## Response Format

API responses are returned in JSON format and include:
- Response data from the SmartAgent
- Conversation metadata
- Processing information
- Error messages (when applicable)

## Error Handling

The API uses standard HTTP response codes:

| Code | Description |
|------|-------------|
| 200 | Success - Request completed successfully |
| 400 | Bad Request - Invalid request parameters |
| 401 | Unauthorized - Invalid or missing authentication token |
| 404 | Not Found - Requested resource doesn't exist |
| 429 | Too Many Requests - Rate limit exceeded |
| 500 | Internal Server Error - Server-side error occurred |

## Rate Limits

Rate limits are configured per organization. Contact your XtendOps teammate for:
- Your current rate limits
- Request rate limit increases
- Best practices for high-volume use cases

## Quick Start Example

Here's a complete example to get you started:

```bash
curl --location 'https://agent-assist.xtendops.com/api/agent-assist' \
  --header 'Authorization: YOUR_API_TOKEN' \
  --header 'Content-Type: application/json' \
  --data-raw '{
    "question": "How do I track my order?",
    "aiAgentId": "your-smartagent-id",
    "conversationId": "unique-conversation-id",
    "humanAgentId": "your-agent-id",
    "responseType": "json",
    "overrides": {
      "inputs": {
        "customer_tier": "premium",
        "order_number": "12345"
      }
    }
  }'
```
